# Vantagens e desvantagens do Node.js

## Vantagens

- Alta eficiência para I/O concorrente.
- Tempo de desenvolvimento rápido.
- Grande ecossistema de bibliotecas.
- Boa integração com aplicações em JavaScript/TypeScript.
- Comunidade ampla e suporte maduro em cloud/container.

## Desvantagens

- Limitação para processamento CPU-bound na thread principal.
- Qualidade heterogênea no ecossistema npm.
- Maior atenção necessária para observabilidade e controle assíncrono.
- Risco de debt técnico com excesso de dependências e pouca padronização.
