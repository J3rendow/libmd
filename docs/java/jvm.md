# JVM — Java Virtual Machine

A JVM é a máquina virtual que executa bytecode Java.

## Componentes

- **Class Loader** — carregamento de classes
- **Runtime Data Areas** — memória (heap, stack, metaspace)
- **Execution Engine** — interpretação e compilação JIT

## Garbage Collection

- Serial GC
- Parallel GC
- G1 GC
- ZGC
